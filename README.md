# online-retail-sql
This SQL project delves into online retail sales using a public dataset that can also be found in the files I have provided. I used SQL to demonstrate revenue performance, customer behavior, product trends, return impact, and overall business health.

Dataset: https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci

Skills Demonstrated:

- SQL aggregations and metric calculations

- Data cleaning and validation

- Customer and revenue impact

- Window functions and ranking

- Time series analysis

- Business-focused problem solving



Skill Examples:

- Revenue contribution from top 10% of customers

- Return rate and revenue loss analysis

- Customer inactivity detection (6-month window)

- Monthly and yearly sales trend analysis

- Product performance analysis by revenue and quantity
